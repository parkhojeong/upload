- spring.servlet.multipart.enabled=true -> DispatcherServlet use MultipartResolver
- if request is multipart, MultipartResolver convert HttpServletRequest to MultipartHttpServletRequest

- MultipartResolver(spring provide) convert HttpServletRequest to StandardMultipartHttpServletRequest(implements MultipartHttpServletRequest)
- so, controller can use MultipartHttpServletRequest instead of HttpServletRequest
  - easy use interface related to multipart file
